# Ejercicios - Conceptos Básicos

## Ejercicio 1: Variables y Tipos de Datos
Crea variables para almacenar:
- Tu nombre (string)
- Tu edad (int)
- Tu altura en metros (float)
- Si tienes mascota (bool)

## Ejercicio 2: Operadores Aritméticos
Dadas las variables `a = 10` y `b = 3`, calcula y muestra:
- La suma
- La resta
- La multiplicación
- La división
- La división entera
- El módulo (resto)
- La potencia (a elevado a b)

## Ejercicio 3: Operadores de Comparación
Compara dos números ingresados por el usuario y muestra:
- Si son iguales
- Si el primero es mayor que el segundo
- Si el primero es menor o igual que el segundo

## Ejercicio 4: Operadores Lógicos
Crea un programa que verifique si un número está entre 10 y 50 (inclusive) usando operadores lógicos.

## Ejercicio 5: Trabajo con Strings
Dada la cadena `texto = "Python es genial"`:
- Muestra la longitud de la cadena
- Convierte todo a mayúsculas
- Convierte todo a minúsculas
- Reemplaza "genial" por "increíble"
- Verifica si contiene la palabra "Python"

## Ejercicio 6: F-strings
Crea un programa que use f-strings para mostrar:
- Tu nombre y edad en una frase
- El resultado de una operación matemática
- Una cadena con formato de moneda (ej: $100.50)

## Ejercicio 7: Entrada y Salida
Crea un programa que:
- Pida al usuario su nombre
- Pida su año de nacimiento
- Calcule y muestre su edad
- Muestre un mensaje personalizado
