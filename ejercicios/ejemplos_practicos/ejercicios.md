# Ejercicios - Ejemplos Prácticos

## Ejercicio 1: Calculadora Mejorada
Crea una calculadora que:
- Permita realizar operaciones básicas (+, -, *, /)
- Tenga un menú para seleccionar la operación
- Permita realizar múltiples operaciones sin salir
- Valide que no se divida por cero

## Ejercicio 2: Lista de Tareas Mejorada
Crea una aplicación de lista de tareas que:
- Permita agregar tareas
- Marcar tareas como completadas
- Eliminar tareas
- Mostrar todas las tareas (pendientes y completadas)
- Guardar las tareas en un archivo de texto

## Ejercicio 3: Contador de Palabras Avanzado
Crea un programa que:
- Cuente palabras en un texto
- Encuentre la palabra más frecuente
- Muestre estadísticas (total de palabras, palabras únicas, promedio de longitud)
- Permita analizar texto desde un archivo

## Ejercicio 4: Gestor de Contactos
Crea un programa para gestionar contactos que:
- Permita agregar contactos (nombre, teléfono, email)
- Buscar contactos por nombre
- Mostrar todos los contactos
- Eliminar contactos
- Guardar contactos en un archivo

## Ejercicio 5: Juego de Adivinanza
Crea un juego donde:
- El programa genera un número aleatorio entre 1 y 100
- El usuario intenta adivinarlo
- El programa da pistas (mayor/menor)
- Cuenta los intentos
- Permite jugar múltiples veces

## Ejercicio 6: Análisis de Notas
Crea un programa que:
- Permita ingresar notas de estudiantes
- Calcule el promedio de cada estudiante
- Determine si aprobó (nota >= 70)
- Muestre estadísticas generales (promedio del grupo, cantidad de aprobados)

## Ejercicio 7: Conversor de Unidades
Crea un conversor que permita:
- Convertir entre diferentes unidades (temperatura, longitud, peso)
- Tener un menú para seleccionar el tipo de conversión
- Validar las entradas del usuario

## Ejercicio 8: Generador de Contraseñas
Crea un programa que:
- Genere contraseñas aleatorias
- Permita especificar la longitud
- Incluya opciones (mayúsculas, minúsculas, números, símbolos)
- Genere múltiples contraseñas a la vez
